# User Manual – Luminor Arc 65 Premium Smart TV

## 1. Product Overview
The **Luminor Arc 65** is a 65-inch flagship OLED television built for cinematic home entertainment.  
It integrates **4K 120 Hz HDR**, **Variable Refresh Rate (VRR)**, and a **NanoShield anti-glare coating** for bright living spaces.

| Specification | Detail |
|----------------|--------|
| Screen Size | 65" OLED |
| Resolution | 3840 × 2160 (4K UHD) |
| Refresh Rate | 120 Hz with VRR |
| HDR Formats | HDR10, HLG, Dolby Vision |
| Brightness | 1000 nits peak |
| Audio | 60 W Dolby Atmos 2.1 |
| OS | Luminor Vision OS 5 |
| Connectivity | Wi-Fi 6, Bluetooth 5.2, HDMI 2.1 × 4, USB × 2 |
| Weight | 21.4 kg (without stand) |

## 2. Safety Guidelines
- Maintain **10 cm** clearance for ventilation.  
- Mount only on **UL-rated wall brackets** supporting **30 kg or more**.  
- If mounting above a fireplace, ensure wall surface temperature < 45 °C.  
- Use the supplied **ARC65-AC adapter** only.  
- Unplug during thunderstorms or long absences.

## 3. Installation & Mounting
### 3.1 Unboxing Dimensions
Box size: **158 cm × 97 cm × 19 cm**, weight **28 kg**.  
Fits most hatchback vehicles when seats are folded.

### 3.2 Mounting Instructions
1. Remove the stand before wall installation.  
2. Align holes with **VESA 400 × 300 mm** bracket.  
3. Use **M6 × 16 mm** screws.  
4. Route cables through the provided rear channels for clean setup.

### 3.3 Placement Tips
- For fireplace setups: use a tilting bracket to maintain viewing angle below 15°.  
- For best color uniformity, view from at least **1.2× screen height** distance.

## 4. Smart Setup
Upon first power-on:
1. Connect to Wi-Fi or Ethernet.  
2. Sign in with a **Luminor ID** or Google account.  
3. Allow automatic firmware updates.  
4. Calibrate remote by pressing *Home + OK* for 3 seconds.

**Voice Control:**  
Use *“Hey Luminor”* to launch apps or adjust settings hands-free.

## 6. Connectivity & Inputs
- **HDMI 2.1 × 4** (eARC on port 1, supports 4K 120 Hz, VRR)  
- **USB 3.0 × 1**, **USB 2.0 × 1**  
- **Optical audio output**  
- **Ethernet LAN**  
- **Wi-Fi 6 / Bluetooth 5.2**  

> **Tip:** For gaming consoles, use HDMI port 1 or 2 for 120 Hz VRR support.

## 7. Software & Updates
**Automatic Updates:** Enabled by default.  
To check manually:  
`Settings → System → About → Firmware Update`.

**Factory Reset:**  
`Settings → System → Reset → Confirm`.

Firmware changelogs are available at  
[https://luminor.tv/arc65/firmware](https://luminor.tv/arc65/firmware).

## 8. Troubleshooting
| Issue | Possible Cause | Solution |
|--------|----------------|----------|
| Reflections in daylight | NanoShield inactive | Switch Picture Mode → Bright Room |
| TV won’t power on | Loose power cord | Reseat power plug and hold Power 5 s |
| HDMI no signal | Wrong port | Use HDMI 1 or 2 for 4K 120 Hz |
| Voice control not responding | Mic muted | Toggle mic switch under Luminor logo |
| Over-fireplace installation concern | Heat rising | Check wall surface temperature < 45 °C |

## 9. Specifications Summary
- Power supply: 100–240 V AC, 50/60 Hz  
- Power consumption: 160 W (typical)  
- Standby: < 0.5 W  
- Operating temperature: 0–35 °C
