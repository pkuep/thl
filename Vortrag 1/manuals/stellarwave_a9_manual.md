# User Manual – StellarWave A9 Quantum Display Monitor

## 1. Product Overview
The **StellarWave A9** is a 32-inch professional display designed for creators and high-performance gaming.  
It combines a 4K 144 Hz quantum-dot IPS panel with adaptive anti-glare optics and factory-calibrated color accuracy.

| Specification | Detail |
|----------------|--------|
| Screen Size | 32" (31.5" viewable) |
| Resolution | 3840 × 2160 (4K UHD) |
| Refresh Rate | 144 Hz (DisplayPort 1.4, HDMI 2.1) |
| Color Coverage | 99% DCI-P3 / 100% sRGB |
| Panel Type | Quantum-dot IPS |
| Brightness | 650 nits typical / 900 nits peak |
| Coating | Adaptive anti-glare matte |
| VESA Mount | 100 × 100 mm |
| Power | 65 W (typical) |

## 2. Safety & Handling
- Avoid exposure to direct sunlight for extended periods.
- Do not cover ventilation openings — the rear cooling vents must remain unobstructed.
- When wall-mounting, ensure the bracket supports at least **10 kg**.
- Disconnect the monitor from power before cleaning. Use a **microfiber cloth only**.

## 3. Setup & Installation
### 3.1 In the Box
- StellarWave A9 display  
- Stand base and neck  
- Power cable  
- DisplayPort 1.4 and HDMI 2.1 cables  
- Calibration report  
- Quick-start card

### 3.2 Mounting
The A9 supports both **desk stand** and **VESA 100 × 100 mm** wall mounts.  
For wall installation:
1. Remove the stand using a Phillips screwdriver.  
2. Align the bracket holes.  
3. Use **M4 × 10 mm** screws (not included).  
4. Maintain at least **10 cm clearance** around the rear vents.

## 4. Connection & Inputs
- **HDMI 2.1 (×2)** — supports 4K @ 120 Hz, VRR (Variable Refresh Rate), HDR10.  
- **DisplayPort 1.4 (×1)** — supports 4K @ 144 Hz with DSC.  
- **USB-C (×1)** — 65 W power delivery, DP Alt Mode, data passthrough.  
- **USB-A hub (×2)** and **audio output (3.5 mm)**.

### Recommended Configuration for Gaming PCs
Use **DisplayPort 1.4** to unlock 4K 144 Hz.  
In Windows → Display Settings → Advanced → select “144 Hz” refresh rate.  
Enable **G-Sync Compatible** or **FreeSync Premium** in your GPU software.

## 5. Display Modes & Calibration
| Mode | Use Case | Characteristics |
|------|-----------|-----------------|
| Creator | Photo & video editing | Factory-calibrated D65, gamma 2.2 |
| Game | Low latency, 144 Hz | Adaptive overdrive, VRR on |
| Cinema | Streaming | Enhanced contrast, HDR10 enabled |
| Eco | Office tasks | Lower brightness, reduced power |

To recalibrate, use the **StellarTune** utility (Windows/macOS) with an X-Rite or Datacolor probe.

## 6. Troubleshooting
| Issue | Possible Cause | Solution |
|--------|----------------|----------|
| Screen flicker at 144 Hz | Cable bandwidth limit | Use DisplayPort 1.4 certified cable |
| Washed-out colors | HDR enabled in OS | Disable HDR for SDR workflows |
| Image ghosting | Overdrive too high | Reduce “Response Time” to *Normal* |
| USB-C no signal | Incompatible laptop | Check DP Alt Mode support |

## 7. Maintenance & Firmware
Firmware updates can be applied via USB or through the **StellarLink app**:
1. Download the `.bin` file from `stellarwave.com/support/a9`.  
2. Copy it to a FAT32 USB drive.  
3. Insert into the USB-A port labeled *Service*.  
4. Select *Settings → Firmware Update → Local USB*.  

During update, **do not power off** the display.
